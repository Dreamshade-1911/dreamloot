You can use hotkeys to make your job easier, check it out:
Tab Select next input box
Ctrl + ↑ Select previous row in the table
Ctrl + ↓ Select next row in the table
Ctrl + M Adds all items from a creature to the list
Ctrl + D Clears the item list
Ctrl + 1-9 Split profit to x amount of people
